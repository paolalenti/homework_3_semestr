# Веб в Java

* `Servlet API` - набор классов для работы с HTTP в Java (стандарт для веб-разработки)
* `war` - `Web Archive` - zip-папка, в которой находятся сервлеты и `web.xml`
* Сервлет (`HttpServlet`) - класс, который способен обрабатывать HTTP-запросы
* `Tomcat` - контейнер сервлетов. Как правило, запускается на порту 8080 и предоставляет возможность для деплоя `war`файлов
* Деплой - распаковка `war` в папку `webapps` и обеспечения доступа к обработке запросов сервлетами этого архива.
